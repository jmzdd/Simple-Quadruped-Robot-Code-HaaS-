# Simple-Quadruped-Robot-Code-HaaS
基于NodeMCU-32S + HaaS四足爬行机器人的简单实现

## 当前仓库
上传了Webots软件的步态仿真工程文件，用的函数也不多，简单的模拟了一下8自由度的四足爬行机器人的步态（蠕动步态）
